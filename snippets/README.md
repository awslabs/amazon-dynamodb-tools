# Snippets / Simple Utilities

This directory contains focused, single-purpose utilities that solve specific problems.

## Current Utilities

- **item_size_calculator** - NPM package for calculating DynamoDB item sizes

## About Snippets

Utilities in this directory are characterized by:
- **Single purpose**: Focused on solving one specific problem
- **Minimal dependencies**: Lightweight with few external requirements
- **Easy to use**: Simple API or command-line interface
- **Well-documented**: Clear usage examples and documentation

## Criteria for Snippets

A tool belongs in snippets if it:
- Solves a specific, well-defined problem
- Can be used independently without complex setup
- Is small enough to be easily understood and maintained
- Doesn't require ongoing infrastructure or services
